# Spin Correlations Comparison

A concrete comparison between the [ALEPH spin-correlation measurement](https://www.sciencedirect.com/science/article/pii/S0370269397006643) and the variable of interest in the latest version of [TauSpinnerTool](https://gitlab.cern.ch/toprocte/TauSpinnerTool/-/tree/zcp-newtauola?ref_type=heads). 

## Environment Installation 

```
python -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt
pip install ipykernel
```
